# Tech_Blog

 ![License](https://img.shields.io/badge/Licence-MIT-red)

  ## About Project

This is a CMS-style blog site  where developers can publish their blog posts and comment on other developersâ€™ posts as well. App is following the MVC paradigm in its architectural structure, and uses Handlebars.js as the templating language, Sequelize as the ORM, and the express-session npm package for authentication. It was developed over some starter code and deployed to Heroku.
